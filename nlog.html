What is the difference between innerHTML, innerText, and textContent ?

innerHtml returns all texts,html tags.textContent is all text contained by an element and all its children for formating purposes only. innerText returns all text contained by an element and all its child elements.

What is event delegation in the DOM?

its a techqnique where we attach a single event listener to a parent element which allows it to handle events that are triggered on its child elements instead of adding seperate listeners to each child element.

What is event bubbling in the DOM?

It is a type of dom propagation where event first triggers on the innermost target element,it also triggers its parent elements in dom hierachy.

How do you create, add, and remove elements using JavaScript?
